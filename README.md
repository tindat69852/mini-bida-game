# Mini Billiard Pocket Game

Mini game HTML5 Canvas đơn giản dùng cho portfolio/GitHub.

## Cách chạy

Mở file `index.html` bằng trình duyệt.

Hoặc dùng VS Code:
1. Cài extension Live Server
2. Chuột phải `index.html`
3. Chọn `Open with Live Server`

## Cách chơi

- Kéo chuột từ gần bi trắng để ngắm.
- Thả chuột để bắn.
- Mỗi bi màu vào lỗ được 10 điểm.
- Đưa hết bi màu vào lỗ để thắng.

## Công nghệ dùng

- HTML
- CSS
- JavaScript
- Canvas 2D
- Collision detection đơn giản
- Basic physics: velocity, friction, wall bounce
